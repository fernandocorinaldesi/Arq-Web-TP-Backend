Bienvenido a microondas-API

La idea es que esta API sea consumida por el front de practica-microondas para generar un muro de publicaciones.
Las publicaciones deberan tener la siguiente estructura :
{

    "Titulo": "Programción 1",

    "Contenido": "Aprender a Programar en lenguaje C....",  

    "Imagen": "path",

    "Usuario": "Fernando",

}

API endpoints

Ruta Descripcion
/login Endpoint para autenticarse con usuario y contraseña
/register Endpoint para registrar un nuevo usuario
/publicaciones Endpoint para crear,modificar o borrar publicaciones
